# 課題管理ツール
Claudeの指示に従って初めて作りました。
大学の課題と締め切りをコマンドラインで管理するPythonツールです。

## 機能
- 課題の追加（課題名・科目・締め切り日）
- 締め切りが近い順に一覧表示
- 優先度表示
- 科目でフィルタリングする
- 残り日数の自動計算
- 課題の検索
- 課題の編集
- 課題の完了マーク
- 課題の削除
- 締め切りアラート
- CSVファイルへの自動保存
- Webアプリ

## 使い方
```bash
python kadai.py
```

## 使用技術
- Python 3.13
- csvモジュール
- datetimeモジュール
